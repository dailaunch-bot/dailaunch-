# ⚙️ Installation

## Clone the Repository

```bash
git clone https://github.com/dailaunch-bot/dailaunch-
cd dailaunch
```

## Install & Build



```powershell
npm install
npm run build:all
npm install -g .\packages\cli
```



```bash
npm install && npm run build:all && npm install -g ./packages/cli
```



```bash
chmod +x setup.sh && ./setup.sh
```



## Verify Installation

```bash
dailaunch --version
# 1.0.0

dailaunch --help
```

You should see all available commands listed.

## Update CLI

To update to the latest version:

```bash
cd dailaunch
git pull
npm run build:all
npm install -g .\packages\cli   # Windows
# or
npm install -g ./packages/cli   # Linux/macOS
```
