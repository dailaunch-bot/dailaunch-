Before installing DaiLaunch, make sure you have the following tools installed.

## Required

### Node.js v18+

Download from [nodejs.org](https://nodejs.org)

```bash
# Verify version
node --version
# v18.x.x or higher
```

### GitHub CLI (`gh`)

Download from [cli.github.com](https://cli.github.com)

```bash
# Verify installation
gh --version

# Login
gh auth login
```

### Git

Download from [git-scm.com](https://git-scm.com)

```bash
# Verify installation
git --version
```

## Optional

| Tool | Purpose |

| **pnpm** | Alternative package manager (faster installs) |
