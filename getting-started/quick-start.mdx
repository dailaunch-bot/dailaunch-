# üöÄ Quick Start

Get your first token live on Base Mainnet in under 2 minutes.

## 1. Login to GitHub

```bash
gh auth login
```

Select: **GitHub.com** ‚Üí **HTTPS** ‚Üí **Login with a web browser**

## 2. Clone & Install

```bash
git clone https://github.com/dailaunch-bot/dailaunch
cd dailaunch
```

**Windows (PowerShell):**
```powershell
npm install
npm run build:all
npm install -g .\packages\cli
```

**Linux / macOS:**
```bash
npm install && npm run build:all && npm install -g ./packages/cli
```

## 3. Deploy Your Token

```bash
dailaunch deploy
```

Fill in the prompts:

| Field | Example | Required |
|-------|---------|----------|
| Token Name | `My Awesome Token` | ‚úÖ |
| Symbol | `MAT` | ‚úÖ |
