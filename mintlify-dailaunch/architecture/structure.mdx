## Tech Stack

| Layer | Technology |
|-------|-----------|
| Blockchain | Base Mainnet (EVM L2) |
| Smart Contract | Clanker SDK v4 |
| Backend API | Node.js + Express + TypeScript |
| Database | PostgreSQL + Prisma ORM |
| CLI | Commander.js + Inquirer |
| Dashboard | Next.js |
| Auth | GitHub OAuth via `gh` CLI |
| Encryption | AES-256 for creator private keys |
| Hosting | Railway / self-host via Docker |
